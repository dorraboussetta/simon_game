# Simon Game Practice Project

## About the Project

This is a guided Udemy practice project for building an interactive Simon memory game.

The game displays four colored buttons. The player starts the game by pressing any key. The game then shows a color sequence, and the player must repeat the sequence by clicking the correct buttons in order. Each correct round adds a new color to the pattern. If the player clicks the wrong color, the game ends and can be restarted.

The HTML and CSS starter files were provided by the instructor. My contribution was writing the JavaScript functionality in `game.js`.
